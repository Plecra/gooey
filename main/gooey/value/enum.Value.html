Enum gooey::value::Value

source · 
pub enum Value<T> {
    Constant(T),
    Dynamic(Dynamic<T>),
}
Expand description

A value that may be either constant or dynamic.

Variants§

§

Constant(T)

A value that will not ever change externally.

§

Dynamic(Dynamic<T>)

A value that may be updated externally.
