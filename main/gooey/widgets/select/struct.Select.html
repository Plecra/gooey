Expand description

A selectable, labeled widget representing a value.

Fields§

§value: T

The value this button represents.

§state: Dynamic<T>

The state (value) of the select.

§kind: Value<ButtonKind>

The button kind to use as the basis for this select. Selects default to ButtonKind::Transparent.
